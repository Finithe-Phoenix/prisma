# docs — Documentación técnica

## Estructura

```
docs/
├── research_notes.md   # Notas de investigación (entregable Fase 0)
├── rfc/                # Request For Comments (decisiones arquitectónicas)
├── blog-drafts/        # Drafts de blog posts públicos
└── reading-notes/      # Notas detalladas por fuente leída (si es muy larga para meterla en research_notes.md)
```

## Convenciones

- **RFCs** numerados: `rfc/0001-ir-ssa-vs-tree.md`, `rfc/0002-tso-classifier-features.md`, etc.
- Cada RFC: Context → Considered Alternatives → Decision → Consequences (reversible?).
- **Blog drafts** se publican en `https://prisma-emu.dev/blog` (futuro). Cadencia target: 1 post cada 2-3 meses desde Fase 0.

## Documentos clave

- [research_notes.md](research_notes.md) — documento vivo con la investigación continua. Entregable primario de Fase 0.
