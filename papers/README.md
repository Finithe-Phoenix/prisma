## Toolchain

- **LaTeX** (Overleaf para colaboración + local `pdflatex`).
- **Bibliografía**: BibTeX con Zotero como source of truth.
- **Figuras**: TikZ para diagramas, matplotlib para data.

## Reglas

- Un paper NO se submit sin co-reviewer externo (contactar académicos en programa committee de conferencia objetivo).
- Datos reproducibles: scripts en `tools/benchmarks/` deben reproducir todas las mediciones.
- Open access vía arXiv preprint incluso si conferencia no exige.
